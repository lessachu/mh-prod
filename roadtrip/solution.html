You'll then need to download the Zappar AR application and scan the Zappar app icon for this puzzle:

At each of the locations, search for each author on loan there: zoom in on the room number plaque via the Zappar app. Looking around in Zappar, you'll find the following emoji which clue a book by another author. Selecting that author in the puzzle page produces a pixel map. Each emoji sequence appears in a virtual location in Zappar (indicated in parentheses in the table below). Collate all the pixel images according to these grid locations.
